# DPay (Deazy Pay) - Crypto & VTU Platform

**DPay** is a high-performance fintech solution designed to make cryptocurrency exchange and virtual top-up (VTU) services simple, secure, and accessible.

## 🛠️ Technology Stack
### Frontend
* **Framework**: Next.js (App Router) & TypeScript.
* **Styling**: Tailwind CSS.
* **State Management**: TanStack Query & Zustand/Context API.
* **Visuals**: Recharts/Chart.js and WebSockets for real-time updates.

### Backend
* **Runtime**: Node.js & Express.js.
* **Database**: MongoDB.
* **Architecture**: RESTful APIs & WebSocket support.

## 🚀 Core Features
### 1. User Dashboard
* **Portfolio Summary**: Dashboard overview with real-time charts and balances.
* **Crypto Wallet**: Secure deposit (wallet address & QR code), withdrawal, and transaction tracking.
* **Conversion Engine**: Live exchange rate integration with conversion estimation.

### 2. VTU Services (Airtime & Data)
* **Automated Top-ups**: Integration with major network providers for instant airtime and data purchase.
* **Tracking**: Transaction tracking and real-time delivery status.

### 3. Admin & Security
* **Admin Panel**: Global overview, user management, and KYC review workflows.
* **Security**: Two-Factor Authentication (2FA), JWT authentication, and data encryption.
